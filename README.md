# Microsoft-office
### My config of microsoft office deployment tool
---
Как установить:
- [Скачайте файлы⏬](https://github.com/shash29exe/Microsoft-office/releases/download/ver1/office.zip) 
- Распакуйте файлы в корень диска C:\, Чтобы получился такой путь: C:\office\SETUP.bat
- Запустите от имени администратора файл SETUP.bat
- Ожидайте завершения установки
---
Есть нюанс:
спустя время слетает активация и офис переходит в режим read-only. На каких то пк у меня получилось активировать, на каких то не поддаётся активации.
